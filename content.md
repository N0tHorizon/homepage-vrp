Welcome to a simple customizable page. Edit this file (content.md) to change the content of the site. Put special directives at the very top on lines that start with an asterisk (*) to change styling quickly.

- To use the dark theme: add *#Dark as the first line.
- To use the light theme: add *#Light as the first line.
- You can override colors or fonts: e.g. *accent:#ff6600 or *bg:#f7f7f8 or *text:#222222
